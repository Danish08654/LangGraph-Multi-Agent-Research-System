# AI-powered autonomous research assistant

The system uses multiple AI agents working together to generate structured research reports on any topic through planning, analysis, writing, and fact-checking workflows.

---

# Features

Multi-agent research pipeline

Planner → Searcher → Analyst → Writer → Fact-Checker workflow

Local LLM inference using Ollama + LLaMA 3

FastAPI backend

Streamlit frontend

Modular LangGraph architecture

Real-time research generation

Offline-compatible setup
